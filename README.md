# San Francisco 2026 Trip

A planner-style family trip dashboard for Washington, DC to San Francisco in August 2026.

## Live site

Once GitHub Pages finishes deploying, the site should be available at:

`https://zapantaarvin.github.io/San-Fran-2026/`

## What the site shows

- a dashboard-style carousel with `Flights`, `3BR Stay`, and `Plan` tabs
- flight guidance with `no Frontier` and `land by 5 PM` rules
- three ranked 3-bedroom rental picks near SFO: `Best`, `Nicest area`, and `Cheapest`
- direct booking links for Airbnb and Vrbo
- a compact Aug 6 to Aug 10 trip plan that protects family time after 4 PM when possible

## Project files

- `index.html` contains the planner dashboard
- `styles.css` contains the planner visual design
- `script.js` powers the tab carousel interaction
- `.github/workflows/pages.yml` deploys the site to GitHub Pages

## Notes

Flight pricing, rental pricing, and travel times can change quickly. Use the booking links on the page to verify before you book.
