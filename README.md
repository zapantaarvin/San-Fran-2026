# San Francisco 2026 Trip

A booking-style family trip planning site for comparing Washington, DC to San Francisco flight options in August 2026.

## Live site

Once GitHub Pages finishes deploying, the site should be available at:

`https://zapantaarvin.github.io/San-Fran-2026/`

## What the site shows

- estimated family totals for the split itinerary
- flight duration and schedule windows for the main options
- booking links to Google Flights, KAYAK, and schedule reference pages
- carry-on guidance so low headline fares do not mislead the comparison

## Project files

- `index.html` contains the flight comparison page
- `styles.css` contains the booking-site-inspired visual design
- `.github/workflows/pages.yml` deploys the site to GitHub Pages

## Notes

The totals are planning estimates built from current route-level fare signals and baggage rules. Final prices and exact times should always be confirmed from the booking links on the page before purchase.
