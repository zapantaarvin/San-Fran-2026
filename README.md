# San Francisco 2026 Trip

A polished family trip planning site for comparing Washington, DC to San Francisco flight options in August 2026.

## Live site

Once GitHub Pages finishes deploying, the site should be available at:

`https://zapantaarvin.github.io/San-Fran-2026/`

## Project files

- `index.html` contains the trip comparison page
- `styles.css` contains the visual design and responsive layout
- `.github/workflows/pages.yml` deploys the site to GitHub Pages

## Notes

The flight totals on the page are planning estimates based on current route-level fare signals and carry-on rules. They are meant for family comparison before final booking.
